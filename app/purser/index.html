  1. SERVICES TO BE PERFORMED

University shall provide the following DataONE services (hereinafter the “Services”):

  1. DataONE Plus

    DataONE Plus includes centralized data search and data management services for creating custom, network-wide scientific data portals spanning DataONE member repositories.

  2. DataONE Hosted Repository

    The Hosted Repository service is a mechanism to affordably operate science data repositories for clients using our open source software and hosted on our computing servers.

  3. High Availability Storage

    Distributed data storage replicated in 3 data centers for use by those using the Hosted Repository service. Add on for DataONE hosted repository.

  4. Data Curation Services

    Data Curation services, including metadata review, creation, and authoring, data modeling, quality assessment, and related curation functions for data in Hosted Repositories, DataONE Plus data portals, and other locations.

  5. Design and Development services

    Design and Development services to extend and customize DataONE Plus data portals and Hosted Repositories.
